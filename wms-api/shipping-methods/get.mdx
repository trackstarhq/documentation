### Response

<ResponseField name="data" type="list of shipping methods">

A list of shipping methods. See the [Shipping Method definition](/wms-api/shipping-methods/info) for more details.

<Expandable title="Shipping Method">

<ResponseField name="id" type="string">
  The unique code of the shipping method.
</ResponseField>

<ResponseField name="created_date" type="string">
  The date the shipping method was created. Returned in YYYY-MM-DDTHH:MM:SSZ format.
</ResponseField>

<ResponseField name="updated_date" type="string">
  The date the shipping method was last updated. Returned in YYYY-MM-DDTHH:MM:SSZ format.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the shipping method.
</ResponseField>

<ResponseField name="description" type="string or null">
  Additional information provided about the shipping method
</ResponseField>

</Expandable>

</ResponseField>
