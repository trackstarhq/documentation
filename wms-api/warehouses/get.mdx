### Response

<ResponseField name="data" type="list of warehouses">

A list of warehouses. See the [Warehouse definition](/wms-api/warehouses/info) for more details.

<Expandable title="Warehouse">

<ResponseField name="id" type="string">
  The unique code of the warehouse.
</ResponseField>

<ResponseField name="created_date" type="string">
  The date the warehouse was created. Returned in YYYY-MM-DDTHH:MM:SSZ format.
</ResponseField>

<ResponseField name="updated_date" type="string">
  The date the warehouse was last updated. Returned in YYYY-MM-DDTHH:MM:SSZ format.
</ResponseField>

<ResponseField name="name" type="string">
  The warehouse name.
</ResponseField>

<ResponseField name="code" type="string or null">
  The warehouse code.
</ResponseField>

</Expandable>

</ResponseField>
