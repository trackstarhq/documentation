---
title: "Get All Inventory"
api: "GET /wms/inventory"
description: "Get all inventory items"
---

### Headers

<ParamField header="x-trackstar-api-key" type="string" required>
  Your organization-level Trackstar API Key.
</ParamField>

<ParamField header="x-trackstar-access-token" type="string" required>
  Your user's access token for a specific integration (ShipHero, Extensiv, etc).
</ParamField>

### Query Params
<ParamField query="page_token" type="string" optional>
  The Trackstar API returns 250 items by default.
  If you make a query that returns more than 250 items, the API will return a `next_token` that you can pass in to get the next page of results.
  Leave this field out to get the first page.
</ParamField>

### Response

<ResponseField name="data" type="list of object">

A list of inventory items.

<Expandable title="Inventory Item">

<ResponseField name="id" type="string">
  The unique ID of an inventory item.
</ResponseField>

<ResponseField name="name" type="string">
</ResponseField>

<ResponseField name="active" type="boolean">
</ResponseField>

<ResponseField name="onhand" type="integer">
  The total amount of inventory in the warehouse.
</ResponseField>

<ResponseField name="committed" type="integer">
  Inventory that is already assigned to orders.
</ResponseField>

<ResponseField name="fulfillable" type="integer">
  Inventory that can be fulfilled for orders.
</ResponseField>

<ResponseField name="exception" type="integer">
  Inventory that is in exception status - quarantined, broken, etc.
</ResponseField>

<ResponseField name="unknown" type="integer">
  Inventory that cannot be sold for another reason - reserved, etc.
</ResponseField>

<ResponseField name="sellable" type="integer">
  Inventory that can be sold.
</ResponseField>

<ResponseField name="awaiting" type="integer">
  Inventory that the warehouse is awaiting delivery of.
</ResponseField>

<ResponseField name="inventory_by_warehouse_id" type="dictionary">

  Inventory broken down by warehouse.

  <Expandable title="Inventory by Warehouse">

  <ResponseField name="warehouse_id_as_key" type="string">

    Warehouse ID.

    <Expandable name="warehouse_info" type="object">

        <ResponseField name="onhand" type="integer">
            Total quantity in the warehouse.
        </ResponseField>

    </Expandable>

  </ResponseField>

  </Expandable>

</ResponseField>

</Expandable>

</ResponseField>

<ResponseField name="next_token" type="string or null">
  The Trackstar API returns 250 items by default.
  If you make a query that returns more than 250 items, the API will return a `next_token` that you can pass in to get the next page of results.
  Otherwise, this will be `null`.
</ResponseField>
