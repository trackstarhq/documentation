---
title: "Exchange Auth Code"
api: "Post https://production.trackstarhq.com/link/exchange"
description: "Exchanges an authorization code for a permanent access token"
---

### Headers

<ParamField header="x-trackstar-api-key" type="string" required>
  Your organization-level Trackstar API Key.
</ParamField>

### Query Params
<ParamField query="auth_code" type="string" required>
  The authorization code returned from the frontend after a user installs an integration.
</ParamField>

### Response

<ResponseField name="access_token" type="string">
  The permanent access token for the integration.
</ResponseField>

<ResponseField name="integration_name" type="string">
  The name of the integration that was installed.
</ResponseField>


<ResponseExample>

```json Response
{
  "access_token": "your_permanent_access_token",
  "integration_name": "installed_integration_name"
}
```

</ResponseExample>
