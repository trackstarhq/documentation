---
title: "Create Link Token"
api: "Post https://production.trackstarhq.com/link/token"
description: "Creates a link token that is required to initiate the react-trackstar-link component"
---

### Headers

<ParamField header="x-trackstar-api-key" type="string" required>
  Your organization-level Trackstar API Key.
</ParamField>


### Response

<ResponseField name="link_token" type="string">
  The link token that is required to initiate the react-trackstar-link component.
</ResponseField>

<ResponseExample>

```json Response
{
  "link_token": "your_link_token"
}
```

</ResponseExample>
