---
title: "Get Connections"
api: "GET /connections"
description: "Returns connected integrations for your organization."
---

### Headers

<ParamField header="x-trackstar-api-key" type="string" required>
  Your organization-level Trackstar API Key.
</ParamField>

### Query Params
<ParamField query="access_token" type="string">
  An optional access token to filter connections by. If included, a single connection associated with the access token will be returned.
</ParamField>

### Response

<ResponseField name="data" type="list of connections">

A list of connections for your organization.
